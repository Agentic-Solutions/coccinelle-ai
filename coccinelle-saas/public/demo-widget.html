🐞 Widget Coccinelle.AI

Intégrez un système de réservation en ligne en 1 ligne de code

Mode Inline Recommandé

Le widget s'affiche directement dans la page, à l'emplacement du script.

<script src="https://coccinelle.app/embed.js" data-coccinelle-tenant="salon_marie" data-position="inline" ></script>

Mode Button

Un bouton flottant en bas à droite qui ouvre une modal.

<script src="https://coccinelle.app/embed.js" data-coccinelle-tenant="salon_marie" data-position="button" data-button-text="Prendre RDV" data-button-color="#000000" ></script>

Mode Popup

Une popup qui apparaît automatiquement après 5 secondes.

<script src="https://coccinelle.app/embed.js" data-coccinelle-tenant="salon_marie" data-position="popup" data-button-color="#667eea" ></script>
